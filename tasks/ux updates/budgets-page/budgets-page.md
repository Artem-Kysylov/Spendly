3. Страница Бюджета (Детальный просмотр)
Цель: Адаптивность. Разделить вид для Десктопа и Мобайла.
 * [ ] Скрыть форму (Mobile): На экранах < md (mobile) полностью скрыть форму добавления транзакции. Вместо нее показать кнопку/FAB "Добавить трату в этот бюджет" (которая открывает глобальную модалку с предзаполненным бюджетом).
 * [ ] Показать форму (Desktop): На экранах >= md оставить форму справа от списка транзакций (как сайд-панель).
 * [ ] Хедер страницы (Mobile):
   * Кнопка "Назад".
   * Название бюджета.
   * Меню "Три точки" (DropdownMenu) справа: внутри пункты "Редактировать" и "Удалить".
 * [ ] Сводная карточка (Hero Card):
   * Показывать крупно "Осталось" или "Накоплено".
   * Прогресс-бар с цветовой кодировкой (Зеленый -> Желтый -> Красный в зависимости от % траты).
